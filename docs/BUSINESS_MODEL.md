**Version**: 2.0 - Multi-Sport Platform

**Status**: Pre-Seed Planning

## 📋 Executive Summary

**비전**: 모든 팀 스포츠의 디지털 락커룸이 되다

**미션**: 아마추어 스포츠 팀의 운영 부담을 제거하고, 추억을 아카이빙하며, 팀원 간 유대감을 강화하는 올인원 플랫폼

**핵심 인사이트**:
- 축구팀이 겪는 문제(출석 관리, 일지 공유, 팀 운영)는 **모든 야외 팀 스포츠의 공통 문제**
- 현재 솔루션(카톡+구글폼+밴드)은 분산되어 있고, 스포츠 특화 UX가 없음
- 멀티스포츠 플랫폼으로 확장 시 **시장 규모 10배 이상 증가** 가능

## 📊 비즈니스 모델 캔버스 (Multi-Sport 버전)

### 1️⃣ 고객 세그먼트

#### Primary Segment (핵심)
- **팀 스포츠 운영진** (Admin)
  - Pain Point: 출석 관리, 일정 공지, 비용 정산, 장비 관리
  - Gain: 자동화로 운영 시간 80% 절감

- **팀 멤버** (User)
  - Pain Point: 분산된 정보, 추억 흩어짐, 참여도 확인 어려움
  - Gain: 한 곳에서 모든 정보 확인, 추억 아카이빙

#### Secondary Segment (확장)
- **B2B**: 스포츠 시설 운영자 (풋살장, 야구장, 테니스장)
  - Pain Point: 예약 관리, 고객 리텐션
  - Gain: 팀 단위 반복 예약, 커뮤니티 형성

- **B2G**: 지자체, 생활체육회
  - Pain Point: 생활체육 활성화, 참여 통계
  - Gain: 데이터 기반 정책 수립

### 2️⃣ 가치 제안

#### 핵심 가치 (All Sports)
1. **운영 자동화**
   - RSVP → 체크인 → 지각비 → 정산 (End-to-End)
   - 푸시 알림 자동 닦달
   - 출석률/참여도 대시보드

2. **추억 아카이빙**
   - 폴라로이드 스타일 피드
   - 게이미피케이션 (MVP, 베스트 플레이)
   - 연말 회고 자동 생성

3. **팀 유대감 강화**
   - 댓글/좋아요/멘션
   - 팀 통계/랭킹
   - 밈/문화 공유

#### 스포츠별 특화 가치
- **축구/야구**: 장비 관리, 유니폼 코디
- **러닝크루**: 기록 관리, 코스 추천
- **테니스**: 복식 페어링, 토너먼트 대진표
- **등산**: 안전 체크인, 날씨 알림

#### 차별화 vs 대체재
| 대체재 | 부족한 점 | 락커룸의 우위 |
|--------|----------|-------------|
| **카카오톡** | 정보 흩어짐, 검색 어려움 | 구조화된 데이터, 아카이빙 |
| **구글폼** | 수동 집계, 실시간 X | 자동 집계, 푸시 알림 |
| **네이버 밴드** | 스포츠 특화 X, 무거움 | 경량화, 스포츠 UX |
| **Teamsnap (해외)** | 한국 UX X, 비쌈 | 한국 문화, Freemium |

### 3️⃣ 채널

#### Customer Acquisition

**Phase 1: Early Adopters (0-500팀)**
- 오프라인: 풋살장/야구장 전단지, QR코드
- 온라인: 네이버 카페/밴드 광고글, 인스타그램 릴스
- 파트너십: 스포츠 인플루언서 협업 (유튜버, 크루 리더)

**Phase 2: Growth (500-2,000팀)**
- B2B 제휴: 풋살장 예약 시스템 연동
- 바이럴 루프: 팀 초대 보상 (무료 기간 연장)
- 페이드 광고: Meta Ads, 네이버 SA

**Phase 3: Scale (2,000팀+)**
- B2G 입찰: 지자체 생활체육 플랫폼
- 크로스 스포츠 추천: 축구팀 → 풋살팀 자동 매칭
- PR: 미디어 노출 (조선일보, SBS 등)

#### Distribution
- **Direct**: 웹앱 (PWA), 추후 네이티브 앱
- **Indirect**: 스포츠 시설 태블릿 설치

### 4️⃣ 고객 관계

#### Acquisition
- **무료 체험**: 첫 3개월 무료 (팀당)
- **온보딩**: 3분 팀 생성 → 첫 운동 등록 → 멤버 초대

- **First Value**: 첫 RSVP 응답 받기 (24시간 내)

- **게이미피케이션**: 출석 스트릭, 팀 랭킹, 시즌 MVP
- **커뮤니티**: 팀 간 친선경기 매칭, 리그 운영

- **팀 초대**: 초대 1명당 +1개월 무료
- **크로스 팀**: 한 유저가 여러 팀 가입 시 할인

#### Revenue
- **Freemium → Premium**: 3개월 후 전환 유도
- **Annual Plan**: 연 결제 시 2개월 할인

## 🎯 Go-to-Market 전략

### Phase 1: Validation (3개월, 0→50팀)

**목표**: PMF 검증, NPS > 50

**전술**:
1. **본인 팀 Dogfooding** (Week 1-2)
   - 본인이 속한 축구팀에서 사용
   - 버그 수정, UX 개선

2. **주변 팀 무료 베타** (Week 3-8)
   - 지인 5개 팀 직접 온보딩
   - 주 1회 피드백 세션

3. **축구 → 풋살/야구 확장** (Week 9-12)
   - 스포츠별 템플릿 추가
   - 각 스포츠당 3개 팀 테스트

**성공 지표**:
- DAU/MAU > 30% (주 1회 운동 팀 → 월 4회 로그인)
- D7 Retention > 60%
- NPS > 50

## 🏆 경쟁 우위 (Sustainable Moat)

### 1. Network Effects (약함 → 강함으로 전환)

**현재 문제**: 한 팀 내에서만 가치 발생 (Single-player mode)

**해결책**: Cross-team Features
- **팀 간 매칭**: 친선경기 상대 찾기
- **리그 시스템**: 지역 리그, 온라인 토너먼트
- **플레이어 풀**: 용병 구인/구직 마켓플레이스
- → **규모 도달 시**: 팀이 많을수록 매칭 품질↑ (Liquidity)

### 2. Data Moat (시간에 비례)

**누적 데이터**:
- 개인 활동 이력 (출석률, 컨디션, MVP 횟수)
- 팀 패턴 (선호 시간대, 장소, 시즌)
- 지역별 스포츠 인사이트

**방어 효과**:
- 개인화 추천 (최적 운동 시간, 팀 매칭)
- 팀 분석 (이탈 위험 예측, 참여 독려)
- → **1년 이상 데이터**: 전환 비용↑

### 3. Switching Costs (높임)

**Lock-in 메커니즘**:
- **추억 아카이빙**: 3년치 사진/일지 → 이동 어려움
- **팀 문화**: 커스텀 배지, 밈, 내부 용어
- **외부 통합**: 시설 예약, 결제 정보 저장

### 4. Product Velocity (속도)

**장점**:
- AI 페어 프로그래밍 (Claude) → 개발 속도 3배
- PWA → 네이티브 앱 없이 빠른 배포
- Multi-tenant → 스포츠 추가 시 코드 재사용

**전략**:
- 경쟁사 대비 **3개월 먼저** 신기능 출시
- 주 1회 업데이트 (사용자 피드백 48시간 내 반영)

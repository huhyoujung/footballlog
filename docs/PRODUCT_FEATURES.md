## 운영 자동화 (운영진 전용)

| 기능 | 설명 |
|------|------|
| **RSVP** | 참석/지각/불참 응답, 마감 시간 설정, 미응답자 자동 리마인더 푸시 |
| **체크인** | 지각 자동 판정, 시간 제한 (이벤트 2시간 전~후) |
| **팀 배정** | 세션별 A/B/C팀 배정, 드래그로 팀원 이동, 배정 완료 시 푸시 알림 |
| **지각비** | 운영진이 부과, PENDING/PAID 상태 관리, 대상자에게 푸시 알림 |
| **장비 배정** | 운동별 장비 담당자 지정 (공, 조끼, 마커콘 등) |
| **조끼 당번** | 로테이션 기반 자동 추천, 순서 수동 조정 가능 |
| **팀 운동 생성/수정/삭제** | 날짜, 시간, 장소, 복장, 메모, POM 투표 설정 포함 |

## 기록 & 일지

| 기능 | 설명 |
|------|------|
| **훈련 일지** | 컨디션(0-10), 컨디션 이유, 핵심 플레이, 개선점, 사진, @멘션 |
| **개인/팀 운동 연결** | 일지를 특정 팀 운동에 연결, 독립 일지도 가능 |
| **일지 수정/삭제** | 본인 일지만 수정/삭제 가능 |

## 피드 & 소셜

| 기능 | 설명 |
|------|------|
| **폴라로이드 피드** | 날짜별 스택, 탭하면 수평 캐러셀 확장 |
| **좋아요** | 토글 방식, 낙관적 UI, 작성자에게 푸시 알림 |
| **댓글** | @멘션 지원(최대 5명), 작성자/멘션된 사람/기존 댓글러에게 알림 |
| **닦달** | "일지 써!" 재촉, 같은 사람에게 1시간 쿨타임, 전광판 노출 |
| **락커 쪽지** | 포스트잇 형태 칭찬 쪽지, 익명 가능, 하루 1장 제한 |
| **전광판 배너** | LED 전광판 스타일, 운동공지/MVP/닦달/활동 메시지 8초 롤링 |
| **POM(MVP) 투표** | 운동 후 2시간~다음날 23:59, 이유 입력 필수, 전광판 노출 |

## 팀 관리

| 기능 | 설명 |
|------|------|
| **초대 코드** | 코드 입력으로 팀 가입, 재생성 가능 |
| **운영진 관리** | ADMIN/MEMBER 역할 변경, 다중 운영진 지원 |
| **팀 설정** | 팀명, 팀 컬러(8가지 프리셋), 로고 |
| **장비 관리** | 장비 CRUD, 팀 고정 담당자 지정 |
| **유니폼 관리** | 유니폼 CRUD, 색상 코드 |
| **출석 통계** | 팀원별 출석률, 지각 횟수, 평균 참석 인원 |

## AI 코칭 (예정)

| 기능 | 설명 |
|------|------|
| **개인 성장 트래킹** | 컨디션 트렌드, 강점/약점 자동 분석, 개선 목표 진척도 |
| **AI 코치 피드백** | 핸드폰 흔들기 → 주간 리포트, 맞춤 훈련 추천 |
| **과거의 나와 비교** | 동료 비교가 아닌 3개월 전 나 vs 지금 나 |
| **팀 인사이트** | 팀 전체 컨디션 트렌드, 공통 이슈 감지, 케미스트리 분석 |
| **환경 요인 분석** | 날씨/구장/시간대별 퍼포먼스 상관관계 |

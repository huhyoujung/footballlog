# AI 인사이트 & 코칭 기능 기획

## 개요
운동일지(TrainingLog) 데이터를 기반으로 개인 성장과 팀 시너지를 분석하는 AI 인사이트 기능

### 핵심 가치
- **개인 맞춤형**: 단순 통계가 아닌 "나만의 성장 곡선" 제공
- **팀 시너지**: 개인뿐 아니라 팀 전체의 패턴과 응집력 분석
- **예측과 추천**: 과거 데이터 기반 컨디션 예측 및 훈련 방법 제안
- **핸드폰 흔들기 인터랙션**: 하루 한 번 AI 코치 피드백 (BM 가능성)

---

## 활용 가능한 데이터

### TrainingLog 데이터
- `condition` (0-10): 객관적 컨디션 점수
- `conditionReason`: 컨디션 이유 (수면, 스트레스, 부상 등)
- `keyPoints`: 오늘의 핵심 포인트 (자유 텍스트)
- `improvement`: 개선하고 싶은 점 (자유 텍스트)
- `notes`: 기타 메모
- `taggedUsers`: 함께 훈련한 팀원 태그

### TrainingEvent 연계 데이터
- 날씨, 구장, 시간대, 체크인 여부, 팀 구성
- MVP 투표, 출석률, 세션 구성

## Tier 1: 개인 성장 트래킹

### 1. 컨디션 트렌드 분석
```
"최근 4주간 컨디션이 평균 7.2점으로 상승세입니다!
특히 화요일 저녁 훈련에서 가장 좋은 컨디션(평균 8.5)을 보이고 있어요."

"12월부터 컨디션이 점진적으로 하락 중입니다(9월 평균 7.8 → 12월 평균 6.2).
주요 이유: '수면부족' 12회 언급. 충분한 휴식이 필요할 것 같아요."
```

**기술 구현**:
- 시계열 데이터 분석 (4주 이동평균)
- conditionReason 텍스트 빈도 분석
- 요일/시간대별 평균 비교

### 2. keyPoints 텍스트 분석 → 강점/약점 자동 파악
```
"최근 10회 운동에서 '패스'를 6회 언급했고, 그 중 4회는 긍정적 평가였습니다.
패스가 당신의 강점으로 자리잡고 있네요!"

"'슈팅' 관련 언급이 8회인데, 모두 개선 필요로 기록되었습니다.
슈팅 정확도 향상을 위한 개인 훈련을 추천드려요."
```

**기술 구현**:
- 자연어 처리 (키워드 추출)
- 감성 분석 (긍정/부정 판단)
- 빈도 분석 및 트렌드 추적

### 3. improvement 추적 → 성장 곡선
```
"지난달 '볼 컨트롤' 개선 목표를 세웠는데,
이번 주 keyPoints에 '볼 터치가 부드러워짐'이 등장했어요!
목표 달성까지 75% 진행 중입니다."
```

**기술 구현**:
- improvement와 keyPoints 상관관계 분석
- 의미론적 유사도 계산 (임베딩 기반)
- 진척도 시각화

### 4. 외부 요인 vs 퍼포먼스 상관관계
```
"당신은 맑은 날씨에 컨디션이 평균 1.3점 높습니다.
다음 주 목요일 훈련은 맑음 예보 - 좋은 컨디션 기대!"

"잠실 구장에서의 평균 컨디션(8.1)이
서울숲 구장(6.8)보다 높아요. 구장 선호도를 팀에 공유해보세요!"

"19시 이후 저녁 훈련에서 컨디션 점수가 평균 0.9점 낮습니다.
충분한 준비운동 시간을 확보하는 것이 좋을 것 같아요."
```

**기술 구현**:
- 다변량 분석 (날씨, 구장, 시간대 vs 컨디션)
- 통계적 유의성 검증
- 예측 모델 (회귀 분석)

## Tier 2: AI 코칭 피드백 (핸드폰 흔들기)

### 5. 주간 AI 코치 리포트
```
[핸드폰 흔들기 → AI 코치 등장]

"이번 주 3회 훈련 참여, 평균 컨디션 7.3점이에요.
keyPoints에서 '드리블 성공률 향상'을 3번 언급하셨네요 - 멋져요!

하지만 improvement에 '왼발 슈팅'이 2주 연속 나타나고 있어요.
다음 훈련 전 왼발 슈팅 연습 10분을 추천드려요."

→ 유튜브 추천 영상 링크 (왼발 슈팅 훈련법)
```

**BM 가능성**:
- 무료: 하루 1회 흔들기
- 프리미엄: 무제한 + 맞춤 영상 추천

### 6. 컨디션 저하 조기 경고
```
"최근 3회 훈련 모두 컨디션 6점 이하예요.
conditionReason에서 '수면부족', '업무 스트레스'가 반복되네요.

다음 훈련은 가볍게 참여하거나,
충분한 휴식 후 복귀하는 것을 권장드려요."
```

### 7. 동료가 아닌 과거의 나와 비교
```
"3개월 전 당신의 평균 컨디션은 6.5점이었어요.
지금은 7.8점 - 20% 상승! 꾸준한 참여가 만든 결과입니다.

당시 improvement 목표였던 '체력'은
지금 keyPoints에서 강점으로 바뀌었어요. 축하해요!"
```

## Tier 3: 팀 차원 인사이트

### 8. 팀 전체 컨디션 트렌드
```
"이번 주 훈련 전 팀 평균 컨디션: 7.2점
지난주(6.8점)보다 상승! 팀 분위기가 좋아지고 있어요."

"목요일 저녁 훈련의 팀 평균 컨디션이 항상 낮습니다(평균 6.1점).
주중 피로도가 누적되는 시점 - 훈련 강도 조절을 고려해보세요."
```

### 9. 팀 공통 이슈 감지
```
"오늘 훈련 후 5명이 keyPoints에 '패스 연결'을 언급했어요.
개인 문제가 아니라 팀 전술적 이슈일 수 있습니다.
→ 다음 훈련: 패스 연계 훈련 추천"

"최근 7일간 4명이 conditionReason에 '날씨'를 언급.
우천 시 대체 훈련장 확보를 검토해보세요."
```

### 10. 팀원 간 케미스트리 분석
```
"운동일지에서 '민수'와 '철수'가 서로를 6회 태그했고,
모두 긍정적 맥락(좋은 패스, 호흡)이었어요.
→ 같은 팀 배치 시 시너지 효과 기대!"

"최근 10회 훈련에서 수비수들이 서로를 많이 태그 중(15회).
수비진 결속력이 강화되고 있네요!"
```

### 11. 팀 전체 강점/약점 집계
```
"지난달 팀 전체 운동일지 분석 결과:
💪 강점: 체력(12명), 패스(9명), 의사소통(8명)
⚠️ 개선 필요: 슈팅(15명), 볼 컨트롤(11명), 수비 위치(7명)

→ 다음 주 세션 추천: 슈팅 정확도 집중 훈련"
```

### 12. 환경별 팀 퍼포먼스
```
"잠실 구장에서 팀 평균 컨디션이 7.8점으로 가장 높아요.
서울숲(6.9점)보다 0.9점 높습니다.
→ 중요한 경기는 잠실 구장 예약 추천!"

"19시 훈련(평균 7.5점)이 21시(6.8점)보다 팀 컨디션 좋음.
→ 가능하면 조금 더 일찍 시작하는 것이 효과적일 수 있어요."
```

### 13. 출석률 vs 팀 컨디션
```
"출석률 80% 이상인 훈련의 팀 평균 컨디션: 7.6점
출석률 60% 이하: 6.2점

→ 많은 인원이 모일수록 서로에게 긍정적 영향!"
```

### 14. 세션 구성 효과 분석
```
"'미니게임' 세션이 포함된 훈련 후
운동일지 keyPoints에 긍정적 단어가 평균 2.3배 많아요.

반면 '체력 훈련'만 했을 때는 improvement에
'체력'이 많이 등장하지만 만족도는 낮음.

→ 체력 + 미니게임 조합 추천!"
```

### 15. 팀 응집력 지수
```
"최근 한 달간 운동일지에서 팀원을 태그한 횟수: 47회 (↑ 23%)
서로를 언급하는 빈도가 늘었어요 - 팀워크 상승 신호!

특히 '좋은 분위기', '재미있었다' 같은 키워드가 3배 증가.
팀 문화가 긍정적으로 변하고 있습니다."
```

### 16. 신규 멤버 적응도 추적
```
"새로 합류한 영희님, 첫 3주간 평균 컨디션 6.5점 →
최근 2주 7.8점으로 상승!

운동일지에서 기존 멤버 3명이 영희님을 태그했고,
영희님도 팀원을 5회 언급 - 성공적으로 적응 중입니다."
```

### 17. 장기 팀 성장 리포트 (분기별)
```
"3개월 전 vs 지금 팀 비교:
- 평균 컨디션: 6.8 → 7.4점 (↑ 0.6)
- 평균 출석률: 72% → 81% (↑ 9%p)
- 운동일지 작성률: 45% → 68% (↑ 23%p)

팀 전체 improvement 달성률:
- 체력: 78% (14명 중 11명 개선)
- 패스: 64% (11명 중 7명 개선)
- 슈팅: 43% (14명 중 6명 개선)

→ 슈팅 훈련 강화 필요!"
```

**핸드폰 흔들기 → AI 코치 등장:**

```
안녕하세요! 이번 주 리포트입니다.

📊 나의 이번 주
- 3회 참여, 평균 컨디션 7.3점
- 강점: 드리블 (3회 언급)
- 개선 중: 왼발 슈팅 (2주 연속 목표)

👥 우리 팀 이번 주
- 팀 평균 컨디션 7.1점 (지난주 +0.4)
- 출석률 85% (역대 최고!)
- 공통 이슈: 5명이 '수비 포지션' 언급
  → 다음 훈련에서 수비 위치 세션 추천

🎯 다음 훈련 팁
- 당신: 왼발 슈팅 연습 10분
- 팀: 수비 포지션 확인 + 미니게임
- 날씨: 맑음 예보 (팀 평균 +0.8점 기대)

계속 달려봐요! 🔥
```

## 기술 스택 (예상)

### AI/NLP
- **텍스트 분석**: OpenAI GPT-4 / Claude API
- **키워드 추출**: TF-IDF, 한국어 형태소 분석 (KoNLPy)
- **감성 분석**: 한국어 감성 사전 또는 사전학습 모델
- **임베딩**: OpenAI Embeddings / Sentence Transformers

### 데이터 분석
- **통계 분석**: 기본 집계 (Prisma aggregations)
- **시계열 분석**: 이동평균, 트렌드 분석
- **상관관계**: 피어슨 상관계수, 다변량 회귀

### 비용 최적화
- **Batch 처리**: 매일 자정 집계 및 분석
- **캐싱**: 주간/월간 리포트 사전 계산
- **Tier 분리**:
  - 무료: 기본 통계 + 주 1회 AI 리포트
  - 프리미엄: 실시간 분석 + 무제한 AI 코치

## 구현 우선순위 (MVP)

### Phase 1: 기본 통계
1. 개인 컨디션 트렌드 (4주 이동평균)
2. 팀 평균 컨디션 대시보드
3. conditionReason 빈도 분석

### Phase 2: 텍스트 분석
4. keyPoints/improvement 키워드 추출
5. 강점/약점 자동 분류
6. 개선 목표 진척도 추적

### Phase 3: AI 코치
7. 핸드폰 흔들기 인터랙션
8. 주간 AI 리포트 생성
9. 맞춤형 훈련 추천

### Phase 4: 고급 분석
10. 팀원 케미스트리 분석
11. 환경 요인 상관관계
12. 예측 모델 (컨디션 예측)
